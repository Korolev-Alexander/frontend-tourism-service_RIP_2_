# Курс по РИП
Репозиторий содержит реализацию клиентских приложений для управления заявками на умные устройства.  
Проект включает:  
- SPA на **React + TypeScript** с маршрутизацией, Redux Toolkit и генерацией API из Swagger  
- Адаптивный интерфейс и PWA, развёрнутые на **GitHub Pages**  
- Нативное приложение на **Tauri** (Windows/macOS/Linux) с подключением к бэкенду по IP  
- Итоговый фронтенд — интерфейс модератора с отображением асинхронных результатов
